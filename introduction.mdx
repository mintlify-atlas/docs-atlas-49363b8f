# Welcome to MatriculaUniPoo

MatriculaUniPoo is a comprehensive desktop application built with Java Swing that streamlines university course enrollment and student registration processes. Developed as a final project for Object-Oriented Programming (POO), it demonstrates practical implementation of OOP principles in a real-world academic management system.

## Available Courses

The system supports enrollment in five core university subjects:

- **Mathematics** - Multiple sections with different professors and schedules
- **History of Peru** - Various time slots throughout the week
- **Literature** - Morning and afternoon options available
- **Civics** - Evening and weekend schedules
- **Sciences** - Flexible scheduling across the week

## Technology Stack

- **Java 20** - Modern Java platform with latest language features
- **Java Swing** - Rich desktop GUI framework
- **Maven** - Dependency management and build automation
- **NetBeans AbsoluteLayout** - Flexible UI layout management

## Use Cases

MatriculaUniPoo is designed to handle the complete student enrollment workflow:

1. **New Student Registration** - Students create accounts with personal information
2. **Secure Authentication** - Login using DNI and password credentials
3. **Course Selection** - Browse available courses with professor and schedule details
4. **Enrollment Management** - Add, remove, and finalize course selections
5. **Schedule Visualization** - View enrolled courses in a clear table format

<Note>
This application was developed as a final project for Object-Oriented Programming, demonstrating practical implementation of OOP concepts including encapsulation, inheritance, and GUI event handling.
</Note>
